4. Security & Compliance

Encryption

TLS 1.3 on all connections. Same encryption as banks. No plaintext ever transmitted.

Authentication

OAuth 2.0 — we never see passwords. Clients authorise specific access, can revoke anytime.

Secrets

API keys encrypted at rest in Cloudflare Secrets. Never in code, never in logs.

Tunnel

Outbound only. No open ports, no firewall holes. Client controls what's exposed.

AI Data

Anthropic (Claude) does NOT store or train on API data. Each request is stateless.

Isolation

Each Worker runs in an isolated V8 sandbox. Can't access other Workers' data.

7. Sales Playbook

The Call Flow

[Before the call: run node tools/call-prep.js "their-url" --html]

You: "Hi [name], I'm Jarek from LayerOps. I noticed [specific issue] on your website — [what it means for their business]. I've got a full report. Want me to walk you through it?"

Them: "Sure / What did you find?"

You: [Screen share — run audit live on layerops.tech]
"Let me show you — I'll run it right now..."

[Score appears. Walk through top 3 issues.]

You: "The biggest one is [issue 1]. This means [business impact]."

You: "Let me also show you something — this is a chatbot I built for a plumber."
[Open demo.layerops.tech, type a question]

You: "Imagine your customers getting answers like this at 2am. That's $49 a month. Want me to set one up for you?"

Objection Handling

"I already have a website."
We're not replacing it. We're adding a smart assistant. One line of code and your website catches enquiries 24/7.
"I don't need AI."
Think of it as a smart answering service. When someone asks a question at 10pm, it answers instead of them leaving.
"$49 a month is too much."
How much is one missed job worth? If it catches one enquiry a month, it pays for itself many times over.
"I'll think about it."
No worries. I'll send you the full report. Can I check in next week?
"Can I see it working?"
Absolutely — demo.layerops.tech. Try asking about blocked drains or booking a time.
"What if I want to cancel?"
No lock-in. Month to month. Cancel any time.
"Is my data safe?"
Everything runs on Cloudflare — same infrastructure that protects 20% of the internet. Encrypted, isolated, no data sharing.

8. Outreach & Lead Generation

The Pipeline

Find businesses (scout.js or web search)
  → Audit them (batch-audit.js or audit dashboard)
  → Generate outreach emails (generate-outreach-html.js)
  → Send emails (send-outreach.js)
  → Follow up automatically (follow-up.js — day 3, 7, 14, 21)
  → Track in CRM (crm.html — auto-updates)
  → Prep for call (call-prep.js)
  → Close the deal
  → Onboard (onboard.js) → live in 15 minutes

Email Strategy (from research)

Email 2 (day 3)Plain text look"Just checking you saw my note"

Key Research Findings

  • 50-75 words = highest reply rate
  • Lowercase subject lines outperform by 15-20%
  • 80% of meetings need 5+ contacts
  • Wednesday 9:30-11:30am = best send time
  • Benefits-led copy converts 2x vs features-led

9. Client Onboarding

$49/month Chatbot Widget (15 minutes)

  1. Run node tools/onboard.js
  2. Enter their business details, services, FAQ, contact info
  3. Config saved to Cloudflare KV
  4. Send them the embed script: <script src="https://slug.layerops.tech/widget/slug"></script>
  5. They paste it before </body> on their site
  6. Done — chatbot is live

$299 SEO Quick Fix (2-3 hours)

  1. Run premium audit: node tools/call-prep.js "their-url" --html
  2. Get their CMS login (WordPress, Squarespace, Wix)
  3. Fix top 5 issues: meta tags, headings, OG tags, schema, alt text
  4. Set up Google Business Profile if they don't have one
  5. Send them before/after audit scores

Multi-Team Setup

For businesses with multiple employees (dental practices, trade companies):

  1. During onboarding, add team members with roles and specialties
  2. Each team member can have their own Cal.com calendar
  3. Chatbot routes customers to the right person based on their needs
  4. Lead notifications go to the specific team member (CC to owner)

10. Daily Operations

Morning Routine (10 minutes)

  • Check Gmail for lead notifications and email replies
  • Open CRM dashboard — check for status changes
  • Check Resend dashboard for email delivery issues
  • Check Cloudflare dashboard for any worker errors

Weekly Routine (1 hour)

  • Run follow-up automation: node tools/follow-up.js --send
  • Generate 1-2 blog posts: node tools/write-blog.js "topic" --publish
  • Post to Google Business Profile (share a blog post)
  • Check Google Analytics for traffic trends
  • Scout for new businesses if outreach pipeline is empty
  • Review and respond to any audit form leads

Monthly Routine (2 hours)

  • Send client reports: chatbot conversations, leads captured
  • Run premium audit on own site — check for regressions
  • Review CRM pipeline — conversion rates, follow-up effectiveness
  • Check Search Console for ranking improvements
  • Export CRM data for records
